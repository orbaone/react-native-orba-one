# react-native-orba-one

Official React Native wrapper for Orba One SDK.

## Installation

```sh
npm install react-native-orba-one
```

## Usage

```js
import OrbaOne from "react-native-orba-one";

// ...

const result = await OrbaOne.multiply(3, 7);
```
